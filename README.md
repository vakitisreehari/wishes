# 🎉 Birthday Wishes Website

A beautiful, animated birthday wish website created with love for Apeksha!

## ✨ Features

- 🎈 Animated floating balloons
- 💖 Floating hearts background
- ⭐ Twinkling stars
- 🎊 Confetti animations
- 💫 Multiple text animations (flip, bounce, slide, glow, pulse, tada)
- 🎵 Interactive buttons with ripple effects
- 🌈 Beautiful gradient backgrounds
- 📱 Fully responsive design
- 💝 Romantic and festive atmosphere

## 🚀 Deployment

This website is deployed on multiple platforms:

- **GitHub Pages**: https://vakitisreehari.github.io/wishes/
- **Vercel**: https://wishes-birthday.vercel.app/
- **Render**: https://birthday-wishes.onrender.com/

## 💻 How to Use

Simply open `index.html` in your web browser, or visit any of the deployed links above.

### Interactive Features:
1. **Make a Wish Button** - Creates confetti and shows a surprise message
2. **Play Music Button** - Displays a birthday song
3. **Click Animated Text** - Each text element can be clicked to trigger surprises
4. **Double-click Name** - Easter egg for extra special wishes

## 🎨 Customization

To personalize this for someone else:
1. Open `index.html` in a text editor
2. Replace "Apeksha" with the person's name
3. Modify the messages and wishes as desired
4. Save and refresh
